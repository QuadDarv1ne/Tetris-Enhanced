# CHANGELOG — Tetris Enhanced

Все заметные изменения в проекте будут задокументированы в этом файле.

## [2.0.0] — 2026-02-23

### ✨ Новые возможности

#### 🎮 Система достижений (Achievements)
- Добавлена полноценная система достижений для отслеживания прогресса игрока
- 19 достижений в базовой комплектации:
  - **Линии**: "Первая линия", "Мастер линий", "Легенда линий"
  - **Очки**: "Первые очки", "Охотник за очками", "Мастер очков", "Легенда очков"
  - **Уровни**: "Новичок", "Опытный", "Эксперт"
  - **Комбо**: "Комбо-мастер", "Комбо-легенда"
  - **Спецдостижения**: "Тетрис!", "Мастер тетрисов", "T-Spin!"
  - **Выживание**: "Выживший", "Долгожитель"
  - **Секретные**: "Идеальная игра", "Скоростной демон"
- Уровни редкости: Bronze, Silver, Gold, Platinum, Diamond
- Визуальные уведомления о разблокировке достижений
- Сохранение прогресса в JSON-файл
- Статистика выполнения и проценты завершения

**Файлы:**
- `game/achievements.py` — основной модуль системы достижений
- `game/__init__.py` — инициализация пакета

#### 🎨 Улучшенные UI компоненты
- **AnimatedButton**: Кнопка с плавными анимациями, масштабированием и эффектами
- **ToggleButton**: Кнопка-переключатель с индикатором состояния
- **IconButton**: Кнопка с поддержкой иконок
- **ProgressBar**: Прогресс-бар с анимацией и пульсацией
- **LoadingSpinner**: Анимированный индикатор загрузки
- **Tooltip**: Всплывающие подсказки с задержкой
- **Dropdown**: Выпадающий список с выбором опций
- **ButtonParticles**: Система частиц для кнопок
- **EasingFunctions**: Функции плавности для анимаций (12 вариантов)
- **UIColor**: Расширенная палитра цветов с утилитами

**Файлы:**
- `ui/advanced_components.py` — продвинутые UI компоненты
- `ui/__init__.py` — инициализация пакета

#### 🛠️ Утилиты логирования
- **EnhancedLogger**: Улучшенный логгер с:
  - Автоматическим созданием отчётов об ошибках
  - Статистикой ошибок по типам
  - Контекстным логированием
  - Системной информацией
- **LogContext**: Контекстный менеджер для логирования блоков кода
- **Декораторы**: `@log_function_call`, `@safe_call`
- **Singleton**: Гарантирует единственный экземпляр логгера

**Файлы:**
- `utils/logger.py` — модуль логирования
- `utils/__init__.py` — инициализация пакета

#### ⚡ Монитор производительности
- **PerformanceMonitor**: Отслеживание FPS, CPU, памяти
- **PerformanceMetrics**: Метрики с историей за 60 кадров
- **LRUCache**: LRU-кэш с ограниченным размером и статистикой
- **ResourcePool**: Пул объектов для повторного использования
- **Профилирование**: Замер времени выполнения функций
- **Декораторы**: `@profile_function`, `@limit_fps`

**Файлы:**
- `utils/performance.py` — модуль производительности

### 🔧 Изменения в зависимостях

**Обновлённые версии:**
- pygame >= 2.5.0
- numpy >= 1.24.0
- pillow >= 10.0.0
- psutil >= 5.9.0

**Опциональные:**
- pygame-menu >= 4.2.0 (для будущих улучшений меню)
- pyinstaller >= 6.0.0 (для сборки исполняемого файла)

### 🧪 Улучшения тестирования

- **TestRunner**: Запуск тестов со статистикой
- **TestCategory**: Категории тестов (unit, integration, performance, ui, config)
- **TestResult**: Детальные результаты с временем выполнения
- **5 категорий тестов**:
  - ConfigTests — тесты конфигурации
  - AdaptiveDesignTests — тесты адаптивности
  - UIComponentTests — тесты UI компонентов
  - PerformanceTests — тесты производительности
  - GameLogicTests — тесты игровой логики

**Команды запуска:**
```bash
python run_tests.py                    # Все тесты
python run_tests.py --category unit    # Только unit-тесты
python run_tests.py -q                 # Тихий режим
```
